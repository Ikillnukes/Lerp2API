Sets the odd row offset.

Sets the offset that is added to each odd row of cells. This offset makes the voronoi cells align to a non-grid. Clamps the value to [-m_CellSize, +m_CellSize].

int cVoronoiMap.m_OddRowOffset

The m odd row offset

The constant amount that the cell seeds of every odd row will be offset from the grid. This allows us to have non-rectangular grids. Expected to be between -m_CellSize and +m_CellSize.
