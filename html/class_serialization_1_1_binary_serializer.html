◆ StartSerializing() [1/2]

bool Serialization.BinarySerializer.StartSerializing ( Entry  entry,
int  id 
)
inline

Called when serializing a new object, the Entry parameter may have MustHaveName set when this is true the name must be persisted as is so that the property or field can be set when retrieving the data. If this routine returns TRUE then no further processing is executed and the object is presumed persisted in its entirety

Parameters
entryThe item being serialized
idThe identifier.
Returns
Normally FALSE. True if the object is already fully persisted

Implements Serialization.IStorage.
