Detailed Description

Provides a builder for asynchronous methods that return Threading.Tasks.Task. This type is intended for compiler use only.

AsyncTaskMethodBuilder is a value type, and thus it is copied by value. Prior to being copied, one of its Task, SetResult, or SetException members must be accessed, or else the copies may end up building distinct Threading.Tasks.Task instances.
